MTSDF Font Rendering (FontsHelper.cs)

ArtFrame utilizes Multi-channel Signed Distance Field (MTSDF) font rendering. Unlike standard rasterized bitmap fonts that blur or pixelate when scaled, MTSDF fonts preserve perfectly sharp, vector-like vector edges at any resolution or magnification.

Technical Concept

MTSDF rendering works by storing distance metrics across four channels (Red, Green, Blue, and Alpha) in a pre-generated atlas texture. During rendering, a custom pixel shader (shaders/atlas.fxb) interpolates these distance values to reconstruct smooth glyph boundaries at high performance.

Public Methods

LoadAtlasFont

Loads and registers an MTSDF font atlas into the global font dictionary.

public static void LoadAtlasFont(string fontName, string jsonPath, string texturePath)

  • Parameters:

    • fontName: A unique key to register this font. Pass this string key to UI elements or text drawings.
    • jsonPath: The path to the font atlas JSON metrics file generated by msdf-atlas-gen.
    • texturePath: The path to the companion PNG atlas texture.

  • Example Usage:

public void Setup()
{
    FontHelper.LoadAtlasFont("main", "fonts/main.json", "fonts/main.png");
}

MeasureText

Measures the virtual bounding dimensions of a string of text based on character advance widths and line count, without accounting for pixel padding.

public static Vector2 MeasureText(string fontName, string text, float scale = 1f)
  • Parameters:
    • fontName: The registered name of the target font.
    • text: The string of text to measure.
    • scale: The text scaling factor (defaults to 1f).
  • Returns: A Vector2 representing the virtual boundary (Width, Height).

MeasureTextBounds

Calculates the exact physical boundaries and visual offset of a string of text, accounting for spacing, character shapes, and distance-field padding. 

public static (Vector2 offset, Vector2 size) MeasureTextBounds(string fontName, string text, float scale = 1f)
  • Parameters:
    • fontName: The registered name of the target font.
    • text: The string of text to measure.
    • scale: The text scaling factor.
  • Returns: A tuple:
    • offset (Vector2): The physical displacement of the first glyph relative to the coordinate origin.
    • size (Vector2): The absolute width and height of the visual bounding box on screen.

DrawTextPro

The framework's high-level text rendering routine. It supports rotation, custom scaling, anchoring, and outline strokes.

internal static void DrawTextPro(
    string fontName,
    string text,
    Vector2 position,
    Vector2 origin,
    float rotation,
    float scale,
    Color color,
    float strokeWidth = 0f,
    Color? strokeColor = null)

Argument Functions & Details

  • fontName — The registered key of the font to use.
  • text — The string of text to draw. Supports multi-line strings split by \n escape sequences.
  • position — The absolute pixel coordinate representing the destination location on the screen.
  • origin — The pivot coordinate (in pixels) relative to the text's bounding box. For example, passing MeasureText() / 2f as the origin centers the text on the position coordinate.
  • rotation — The rotation angle of the text, specified in degrees (rotates around the origin pivot).
  • scale — The font size multiplier. MTSDF preserves razor-sharp edges even at extremely large scales (e.g. 10f).
  • color — The master text tint and opacity.
  • strokeWidth — The thickness of the outline stroke in pixels (defaults to 0f for no stroke).

  • strokeColor — The color of the outline stroke. Drawn as an underlying offset matrix layer.

  • Low-Level Manual Drawing Example:

public void ManualDraw(float dt)
{
    string textStr = "ArtFrame Engine";
    float fontScale = 3f;

    // Measure text boundaries
    Vector2 size = FontHelper.MeasureText("main", textStr, fontScale);

    // Center pivot origin
    Vector2 origin = size / 2f;

    // Render at screen coordinates with a black outline stroke
    FontHelper.DrawTextPro(
        fontName: "main",
        text: textStr,
        position: new Vector2(640, 360),
        origin: origin,
        rotation: 0f,
        scale: fontScale,
        color: Color.White,
        strokeWidth: 2f,
        strokeColor: Color.Black
    );
}
